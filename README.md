# terraform_template1
terraform template to create VM's in proxmox cluster
#
This is the template that I used to create VM in proxmox cluster using terraform 
#
# Instructions:
clone the repo to local system
replace all the text details with the original details of VM and clone VM details
run 'terraform init'
use command 'terraform plan'
and then 'terraform apply'
if everying goes well a new VM with mentioned specifications will be created in proxmox cluster
If errors encountered, please check host address, clone vm, network and credentials details carefully
